============
Installation
============


Stable release
--------------

To install bids-statsmodels-design-synthesizer, run this command in your terminal:

.. code-block:: console

    $ pip install bids_statsmodels_design_synthesizer

This is the preferred method to install bids-statsmodels-design-synthesizer, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for bids-statsmodels-design-synthesizer can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/leej3/bids_statsmodels_design_synthesizer

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/leej3/bids_statsmodels_design_synthesizer/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/leej3/bids_statsmodels_design_synthesizer
.. _tarball: https://github.com/leej3/bids_statsmodels_design_synthesizer/tarball/master
